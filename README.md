# Disby

A high-performance, long-term habit consistency tracker.

## Features
- **Lifetime Data Analysis**: Visualize your efficiency over years with spline-interpolated AreaCharts.
- **Categorized Routines**: Manage habits by life area.
- **Fair Efficiency Logic**: Efficiency is calculated from the routine's first completion, not the start of time.
- **Interactive Legends**: Filter and analyze specific routines in real-time.
- **Fullscreen Mode**: Deep-dive into historical data with a high-resolution zoom brush.

## Tech Stack
- React 19
- TypeScript
- Supabase (Backend/Database)
- Recharts (Data Visualization)
- TailwindCSS (Styling)
- Vite (Build Tool)
